# 한글 폰트 설치 가이드

PDF에서 한글이 깨지지 않도록 하려면 이 폴더에 한글 폰트 파일을 추가해야 합니다.

## 폰트 다운로드 방법

### 옵션 1: Noto Sans KR (권장)

1. https://fonts.google.com/noto/specimen/Noto+Sans+KR 접속
2. "Download family" 클릭
3. 압축 해제 후 다음 파일들을 이 폴더에 복사:
   - `NotoSansKR-Regular.ttf`
   - `NotoSansKR-Bold.ttf`

### 옵션 2: 나눔고딕

1. https://hangeul.naver.com/font/nanum 접속
2. 나눔고딕 다운로드
3. 압축 해제 후 다음 파일들을 이 폴더에 복사:
   - `NanumGothic.ttf`
   - `NanumGothicBold.ttf`

## 설치 후

서버를 재시작하면 자동으로 한글 폰트가 적용됩니다.
