# OpenAI 모델 선택 가이드

## 문서 요약 작업에 적합한 모델 비교

### 1. GPT-4o-mini (추천 ⭐)

**현재 사용 중인 모델**

**장점:**
- ✅ **가격이 매우 저렴함** (gpt-4o 대비 약 10배 저렴)
- ✅ **문서 요약에 충분한 품질**
- ✅ **빠른 응답 속도**
- ✅ **한글 지원 우수**
- ✅ **일반적인 비즈니스 문서 요약에 적합**

**단점:**
- ❌ 복잡한 분석은 gpt-4o보다 떨어짐

**가격 (2024년 기준):**
- 입력: $0.15 / 1M 토큰
- 출력: $0.60 / 1M 토큰

**적합한 용도:**
- 일반 문서 요약
- 핵심 포인트 추출
- 간단한 보고서 생성
- **대부분의 문서 자동화 작업에 적합** ⭐

---

### 2. GPT-4o

**고성능 모델**

**장점:**
- ✅ **가장 높은 품질**
- ✅ **복잡한 문서 분석에 우수**
- ✅ **정확한 핵심 포인트 추출**
- ✅ **긴 문서 처리에 강함**

**단점:**
- ❌ **가격이 비쌈** (gpt-4o-mini 대비 약 10배)
- ❌ 응답 속도가 상대적으로 느림

**가격 (2024년 기준):**
- 입력: $2.50 / 1M 토큰
- 출력: $10.00 / 1M 토큰

**적합한 용도:**
- 복잡한 기술 문서 분석
- 법률 문서 분석
- 고품질 보고서 생성
- 정확성이 중요한 경우

**구형 모델 (사용 비추천)**

## 추천 모델

### 일반적인 문서 요약 작업: **GPT-4o-mini** ⭐

**이유:**
1. **가성비가 가장 좋음** - 가격 대비 품질이 우수
2. **문서 요약에 충분한 성능** - 대부분의 비즈니스 문서 요약에 적합
3. **한글 지원 우수** - 한국어 문서 요약에 문제 없음
4. **빠른 응답** - 사용자 경험이 좋음

**현재 코드 설정:**
```typescript
model: "gpt-4o-mini"  // 이미 최적의 선택!
```

---

### 고품질이 필요한 경우: **GPT-4o**

**이유:**
1. 가장 정확한 요약 필요
2. 복잡한 문서 분석 필요
3. 예산이 충분한 경우

**코드 변경:**
```typescript
model: "gpt-4o"  // 더 높은 품질, 더 높은 비용
```

## 실제 사용 예시

### GPT-4o-mini 사용 시

**문서 길이: 10,000자**
- 입력 비용: 약 $0.002 (약 2원)
- 출력 비용: 약 $0.01 (약 10원)
- **총 비용: 약 12원**

**문서 길이: 100,000자**
- 입력 비용: 약 $0.02 (약 20원)
- 출력 비용: 약 $0.01 (약 10원)
- **총 비용: 약 30원**

### GPT-4o 사용 시

**문서 길이: 10,000자**
- 입력 비용: 약 $0.03 (약 30원)
- 출력 비용: 약 $0.10 (약 100원)
- **총 비용: 약 130원** (gpt-4o-mini의 약 10배)

## 권장 설정

### 개발/테스트 단계
```env
# .env.local
OPENAI_TEST_MODE=true
```
- 테스트 모드 사용 (결제 불필요)

### 실제 운영 단계
```env
# .env.local
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4o-mini  # 또는 환경 변수로 설정
```
- GPT-4o-mini 사용 (가성비 최적)

### 고품질이 필요한 경우
```env
# .env.local
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4o  # 환경 변수로 모델 선택 가능
```
- GPT-4o 사용 (더 높은 품질)
